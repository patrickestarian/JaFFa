	FriendFeed Backup - downloads the feeds and any thumbnail, image, or
	other files attached to the feed to your local file system.


THIS DIRECTORY CONTAINS:
------------------------
	ffbackup.jar - the binary java executable
	
	ffbackup-src.zip - compressed source files
	
	GPL.TXT - GNU Public License
	
	REDME.TXT - the project description


REQUIREMENTS:
-------------
	JRE 1.6 (or a greater version)


HOW TO USE:
-----------
	FORMAT:
	
		java -jar ffbackup.jar <user> <path> [proxy-server:port]

	Example 1 (without proxy):
	
		java -jar ffbackup.jar patrickestarian c:/backup/ff

	Example 2:
	
		java -jar ffbackup.jar patrickestarian c:/backup/ff proxy-server:8080
